API

You can also directly use the API to generate a sitemap. To do this, append the URL of the website to the following base URL:

https://getsitemap.funkpd.com/json?url=

For example, to generate a sitemap for "https://funkpd.com", you would use the following URL:

https://getsitemap.funkpd.com/json?url=https://funkpd.com
